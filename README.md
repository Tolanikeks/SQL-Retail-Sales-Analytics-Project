# SQL-Retail-Sales-Analytics-Project
A business-driven SQL analytics project focused on customer behavior, revenue trends, employee performance, and product sales analysis.
## Project Title
## Retail Sales & Customer Analytics Using SQL
## Project Overview

This project analyzes sales transactions from a health and beauty retail company operating across multiple Nigerian cities.

The goal is to use SQL to answer key business questions related to:

Revenue performance
Customer retention
Product performance
Employee productivity
Refund analysis
Operational efficiency

The project demonstrates advanced SQL concepts including:

Joins
Common Table Expressions (CTEs)
Window Functions
Conditional Aggregation
Ranking Functions
Business KPI Analysis

## Problem Statement

The company collects large volumes of sales data but lacks visibility into customer purchasing behavior, product performance, employee productivity, and revenue trends.

Management requires data-driven insights to answer important business questions such as:

Which products generate the most revenue?
Which employees contribute the highest sales?
How many customers return after their first purchase?
How much revenue is lost through refunds?
Which cities generate the highest revenue?
What are the monthly sales trends?

This project was developed to transform raw transactional data into actionable business insights using SQL.
